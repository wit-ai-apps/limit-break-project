# LIMIT BREAK PROJECT

Every Day, Beyond Yesterday.

生徒・保護者・カウンセラー先生が学習ミッションと日々の記録を確認するための検証版アプリです。現在はGitHub Pagesで外部閲覧できる静的アプリとして公開し、進捗保存はブラウザのlocalStorageを使用します。

## 検証版の範囲

- スタディサプリ教材構造に沿ったToday's Mission表示
- DAY / LEVEL / 時間帯 / 講座 / PART・Chapter / 学習フロー表示
- Mission Clear記録
- 初回確認テスト、翌日確認テスト、3日後確認テスト、7日後確認テスト
- 点数、理解度、疲労度、間違えた理由、本人だけの内省メモの保存
- 今日のMission達成率、科目別達成率、残りMission数
- 復習リマインダー
- 仮ログインによる4ロール切替
- CSV出力
- GitHub Pages公開

## ログイン・権限設計

Phase1では仮ログインUIのみを実装しています。将来的にはFirebase Authenticationなどで本人確認を行います。

| Role | 今日のMission | 点数 | 疲労度 | 内省メモ | 週報 | Mission編集 |
|---|---|---|---|---|---|---|
| student | ○ | ○ | ○ | ○ | ○ | × |
| parent | ○ | △ | △ | × | ○ | × |
| supporter | △ | × | × | × | △ | × |
| teacher | ○ | ○ | ○ | ○ | ○ | ○ |

### student

本人用です。今日のMission、確認テスト、点数、理解度、疲労度、AIコメント想定枠を見られます。Mission完了、点数入力、理解度入力、メモ入力ができます。

### parent

保護者用です。学習時間、Mission達成率、確認テスト推移、週間レポート、AIコメント想定枠を見られます。本人だけの内省メモや、過度にプレッシャーになる情報は表示しません。

### supporter

親戚、家庭教師、メンター、応援者用です。Mission達成率、今週の努力量、応援メッセージ欄を中心に表示します。詳細な成績、疲労度、個人メモは表示しません。

### teacher

塾講師用です。全科目の進捗、確認テスト点数、理解度、疲労度、間違い理由、復習漏れ、週間分析を見られます。将来的にはMission調整、復習追加、コメント追加、次週方針作成を行います。

## 将来構成

正式版ではFirebase + Firestoreを使い、生徒・保護者・カウンセラー先生が同じ進捗を確認できる構成に移行します。

内部設計上の最終目標として難関大学受験を見据えますが、生徒画面では日々の目的である「昨日の自分を超える」ことを中心に表示します。
